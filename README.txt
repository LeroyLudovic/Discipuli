make : compile tous les fichiers nécessaire au fonctionnement du programme
make veryclear : supprime tous les fichiers générés par la fonction "make"
./bin/Discipuli.out : exécute le programme

règles du jeu :
Basiquement, empêcher les élèves d’avoir leur diplôme.

Pour cela, vous pouvez poser des professeurs sur le terrain (ailleurs que sur le chemin) en cliquant sur une case de celui-ci, indiquée par un carré jaune.

Si la case est vide, lorsque vous cliquez, le type de professeur sélectionné en bas avec les touches « A », « Z », « E », et « R » est généré sur le terrain.

Si la case n’est pas vide et contient un professeur, ses statistiques s’afficherons en bas à droite, ainsi que la possibilité d’ « Upgrade » pour améliorer ses statistiques, et de « Vendre » pour se débarrasser de ce professeur.

les élèves arrivent par vague de plus en plus grosse et de plus en plus forte.

Contrôles :
  clique gauche : beaucoup de fonctions, tout dépend d'où vous cliquez.
    clique sur "Play" : lance le jeu.
    clique sur "Quit" : ferme le programme.
    clique sur une case vide du terrain : ajoute un professeur.
    clique sur une case avec un professeur : affiche ses statistiques.
    clique sur "Upgrade" : améliore les statistiques du professeur selectionné.
    clique sur "Vendre" : non, on ne fait de traffic d'êtres humains, il est simplement "licencié".
  A, Z, E, R : permet de changer le type de professeur selectionné (indiqué en bas par un carré jaune).
  q : si vous êtes sur le menu, ferme le programme; si vous êtes sur le jeu, vous ramène au menu.
